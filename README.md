# Vs-Code Shortcuts

## Code Editing

- **Move By Word**

```
Ctrl+Right
Ctrl+Left

Cmd+Right
Cmd+Left
```

- **Go To Start & End Of Line**

```
home
end
```

- **Go To Start & End Of File**

```
Ctrl+Home
Ctrl+End

Cmd+Up
Cmd+Down
```

- **Cut, Copy & Past Line**

```
Ctrl+X # Cut
Ctrl+C # Copy
Ctrl+V # Paste

Cmd+X #Cut
Cmd+C #Copy
Cmd+V #Paste
```

- **Move Line Up & Down**

```
Alt+Up
Alt+Down

Option+Up
Option+Down
```

- **Copy Line Up & Down**

```
Shift+Alt+Up
Shift+Alt+Down


Shift+Option+Up
Shift+Option+Down
```

- **Remove Line**

```
Shift+Ctrl+K

ShiftCmd+K
```

- **Insert Line**

```
Ctrl+Enter # Insert below
Shift+Ctrl+Enter # Insert above

Cmd+Enter # Insert below
Shift+Cmd+Enter # Insert above
```

- **Jump To Matching Bracket**

```
Shift+Ctrl+\

Shift+Cmd+\
```

- **Add Line Comment**

```
Ctrl+/

Cmd+/
```

- **Add Block Comment**

```
Shift+Alt+A

Shift+Option+A
```

- **Highlight Code to Any Direction**

```
Shift+Any Direction
```

- **Highlight Code According to Code Block**

```
Shift+Alt+Arrow Right/Left
```

- **Select Next Match**

```
Ctrl+D

Cmd+D
```

- **De-select Match**

```
Ctrl+U

Cmd+U
```

- **Add Multiple Cursor**

```
Alt+Click

Option+Click
```

- **Go to Entity (Functions, vars, etc)**

```
Ctrl+Shift+O

Cmd+Shift+O
```
